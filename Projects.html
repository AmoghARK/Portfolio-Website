Projects

Admission predictor web

Built a predictive model using pipeline infrastructure and multiple linear regression predicting probabilities of procuring admission within threshold and deployed using Streamlit, a web based interactive model.The objective of the Admission Predictor Web Project is to create a web-based application that leverages machine learning algorithms to predict a student's likelihood of gaining admission to an educational institution based on their academic credentials and other relevant factors

Fire Fighting Bot

A Fire Fighting Bot IoT (Internet of Things) project is designed to create an autonomous robot that can detect and extinguish fires in various environments. This project leverages IoT technologies to monitor and control the robot remotely while gathering data from various sensors for efficient fire detection and firefighting.
